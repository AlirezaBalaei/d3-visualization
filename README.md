open git bash and run:

```
git clone https://github.com/AlirezaBalaei/d3-visualization.git
```

then open directory:

```
cd d3-visualization
```

then install modules:

```
npm install
```

then run this shell command:

```
./node_modules/.bin/http-server -a localhost -p 8000 -c-1
```
